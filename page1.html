Data Analysis

Published: June 2025 • Last updated: July 2025

Overview

This project focuses on visualizing housing market trends in King County, Washington, using real-world data. The dashboard offers interactive insights into house sales, including daily average sale prices, price distribution, bedroom and bathroom counts, and geographical patterns. Users can filter the data by month, year, square footage, and more, enabling deep exploration of the housing market dynamics.

Tech Stack

  • Microsoft Excel, SQL
  • Tbaleau frontend
  • Geographic map showing sales distribution by location

Outcomes

Delivered a dynamic and user-friendly Tableau dashboard that provides clear, actionable insights into housing trends in King County. The project helped identify pricing trends over time, regional price disparities, and how property features (like views and condition) correlate with sales prices. This dashboard could be used by real estate analysts, potential homebuyers, or policymakers to make informed decisions.
