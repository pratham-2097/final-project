Yolo: Image Recognition and Object Detection

Published: June 2025 • Last updated: July 2025

Overview

This project focused on enhancing workplace safety using computer vision and drone-based surveillance. By leveraging deep learning techniques through the YOLOv8 object detection framework, the system was designed to recognize and classify safety equipment on workers and detect the presence of individuals in restricted or dangerous zones, such as blast areas. The goal was to automate safety monitoring and reduce the risk of human oversight. The project was developed collaboratively by a team of four and included the creation of an interactive Streamlit dashboard to visualize detection statistics and insights.

Tech Stack

  • Programming Language: Python
  • Deep Learning Framework: YOLOv8
  • Tools & Libraries: OpenCV, NumPy, Pandas, PyTorch
  • Collaboration: Team of four with shared responsibilities in model training, video processing, and UI development

Outcomes

Successfully developed an AI-powered system capable of real-time object detection via drone footage, accurately identifying safety gear and human presence in hazardous areas. The project delivered a working prototype with a responsive Streamlit dashboard that allowed users to track detection metrics and analyze risk zones visually. This system serves as a foundation for deploying AI-driven safety enforcement in industrial and construction environments.
